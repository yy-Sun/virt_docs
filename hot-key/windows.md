## 一、Win 快捷键

| 快捷键       | 操作                                     |
| ------------ | ---------------------------------------- |
| win + L      | 锁屏                                     |
| win + D      | 打开/隐藏桌面                            |
| win + S      | 打开搜索                                 |
| win + 向下键 | 最小化                                   |
| win + 向上键 | 最大化                                   |
| win + E      | 打开文件资源管理器                       |
| win + I      | 打开设置                                 |
| win + 数字   | 快速打开任务栏应用（数字表示第几个应用） |

## 二、ALT 快捷键

| 快捷键       | 操作                 |
| ------------ | -------------------- |
| alt + tab    | 在打开的应用之间切换 |
| alt + F4     | 关闭活动项，退出任务 |
| alt + 向右键 | 前进                 |
| alt + 向左键 | 后退                 |
| alt + F      | 打开文件栏           |

## 三、Ctrl 快捷键

| 快捷键      | 操作             |
| ----------- | ---------------- |
| Ctrl + C    | 选中复制         |
| Ctrl + X    | 选择剪切         |
| Ctrl + V    | 粘贴             |
| Ctrl + Z    | 撤销操作         |
| Ctrl + S    | 保存             |
| Ctrl + F    | 查找             |
| Ctrl + home | 快速移动到文件头 |
| Ctrl + D    | 删除选中项       |
